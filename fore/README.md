# fore

Memo Echo 的前端管理台，使用 Vue 3 + Vite 构建。

## 开发

```bash
npm install
npm run dev
```

## 打包

```bash
npm run build
```

## 页面入口

- `/` 落地页
- `/auth` 登录 / 注册
- `/dashboard` 后台控制台
